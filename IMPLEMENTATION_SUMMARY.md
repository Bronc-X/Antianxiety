### 1. **用户问卷强化 - 代谢健康困扰多选**

#### 修改文件
- `components/HealthProfileForm.tsx`
- `lib/aiAnalysis.ts`

#### 新增功能
在健康参数设置表单的第三步（生活习惯）中添加了**"代谢健康困扰"**多选项：

**5个困扰选项**（基于2024年代谢衰老研究）：
1. 🔋 **容易疲劳 / 精力不足**
   - 机制：线粒体功能障碍
   - 描述：ATP生成减少，可能与久坐或肌肉量不足有关

2. 🫄 **腹部容易长肉**
   - 机制：IL-17/TNF炎症通路
   - 描述：内脏脂肪积累，可能与胰岛素抵抗有关

3. 💪 **肌肉松弛 / 力量下降**
   - 机制：肌少症风险
   - 描述：30岁后每年流失1-2%肌肉量

4. 🏃 **恢复速度慢**
   - 机制：线粒体+氧化应激
   - 描述：运动后需要较长时间恢复

5. 🍚 **对碳水渴望增加 / 餐后困倦**
   - 机制：代谢重编程
   - 描述：燃料偏好从脂肪转向葡萄糖

#### 用户体验
- ✨ 多选卡片式UI，实时显示已选择数量
- ✨ 每个选项显示科学机制和症状描述
- ✨ 选中后提示："已选择 X 项困扰，AI将为您制定针对性干预方案"

### 2. **AI分析逻辑增强**

#### 修改文件
- `lib/aiAnalysis.ts`

#### 新增功能

##### A. 风险因素自动识别
根据用户选择的困扰，自动添加对应的风险因素：

| 困扰 | 自动识别的风险因素 |
|------|-------------------|
| 容易疲劳 | ATP生成减少（线粒体功能障碍） |
| 腹部长肉 | IL-17/TNF炎症通路激活 |
| 肌肉松弛 | 肌少症风险（30岁后每年流失1-2%肌肉量） |
| 恢复慢 | 恢复能力下降（氧化应激） |
| 对碳水渴望 | 代谢重编程（燃料偏好转向葡萄糖） |

##### B. 健康指标调整
根据困扰自动调整相关健康指标评分：
- 容易疲劳 → 代谢率设为 `low`，精力稳定性设为 `unstable`
- 腹部长肉 → 炎症风险设为 `high`，激素平衡设为 `imbalanced`
- 肌肉松弛 → 代谢率 `low`，恢复能力 `low`
- 恢复慢 → 恢复能力 `low`，炎症风险 `high`
- 对碳水渴望 → 精力稳定性 `unstable`，激素平衡 `imbalanced`

##### C. 置信度提升
用户提供代谢困扰信息后，分析置信度+20分，并记录原因。

### 3. **针对性微习惯推荐**

#### 修改文件
- `lib/aiAnalysis.ts` - `generateRecommendationPlan()` 函数

#### 新增功能
根据用户选择的困扰，生成**循证医学支持**的微习惯方案：

#### 🔋 容易疲劳 → Zone 2有氧运动
```
名称：Zone 2有氧运动
提示：每天早晨或下午精力较好时
行动：进行30分钟低心率有氧运动（60-70%最大心率）
频率：每日
科学依据：提升线粒体功能，增加ATP生成。研究显示8-12周可提升基础代谢率5-10%（Cabo et al. 2024）
```

#### 🫄 腹部长肉 → 16:8禁食 + 抗炎食物
```
习惯1：16:8间歇性禁食
提示：每天晚上8点后
行动：停止进食，只喝水或无糖茶，直到次日中午12点
科学依据：抑制IL-17/TNF炎症通路，改善胰岛素敏感性20-30%（Kwon et al. 2019）

习惯2：抗炎食物摄入
提示：每日午餐和晚餐
行动：摄入深海鱼（Omega-3）或绿茶（多酚）
科学依据：降低炎症标志物（CRP、IL-6）20-30%（Izadi et al. 2024）
```

#### 💪 肌肉松弛 → 抗阻训练 + 蛋白质
```
习惯1：抗阻训练
提示：每周一、三、五
行动：进行自重深蹲或俯卧撑（3组×8-12次）
科学依据：每周3-4次抗阻训练可提升基础代谢率5-8%。30岁后每年流失1-2%肌肉量（Chen & Wu 2024）

习惯2：优质蛋白补充
提示：早餐或运动后30分钟内
行动：摄入20-30g优质蛋白（鸡蛋、乳清蛋白）
科学依据：亮氨酸激活mTOR通路，促进肌肉蛋白合成（Deng et al. 2024）
```

#### 🏃 恢复慢 → 主动恢复 + Omega-3
```
习惯1：主动恢复
提示：运动后或疲劳时
行动：进行10-15分钟轻度有氧运动或拉伸
科学依据：促进血液循环，加速代谢废物清除。配合冷热交替浴效果更佳

习惯2：Omega-3补充
提示：每日午餐或晚餐
行动：摄入1-2g EPA+DHA（深海鱼或鱼油）
科学依据：减少氧化应激，改善线粒体功能，降低炎症标志物20-30%
```

#### 🍚 对碳水渴望 → 低GI饮食 + 规律进餐
```
习惯1：低GI饮食
提示：每日三餐
行动：选择低GI食物（全谷物、蔬菜），避免精制碳水
科学依据：稳定血糖，逆转代谢重编程。配合16:8禁食可优化RER（呼吸交换率）

习惯2：规律进餐
提示：每3-4小时
行动：小份餐食，避免长时间空腹后暴食
科学依据：维持能量稳定性，防止血糖波动。短暂户外活动可快速恢复精力
```

### 4. **Pro版功能 - AI甄选抗衰食材**

#### 新增文件
- `components/ProAntiAgingFoods.tsx` (全新组件)

#### 功能亮点

##### 🌟 科学背书
基于 **Arora et al. 2024 (Nature Aging) AgeXtend研究**
- AI/机器学习预测具有抗衰老潜力的天然分子
- 完整DOI引用：10.1038/s43587-024-00763-4

##### 📊 5种AI甄选抗衰分子

**1. 姜黄素 (Curcumin)**
- 机制：抗炎、抗氧化，激活Nrf2通路
- 益处：减少慢性炎症（IL-6、TNF-α ↓30%），改善线粒体功能
- 食物来源：姜黄根、咖喱粉
- 建议剂量：500-1000mg配黑胡椒（提升吸收2000%）
- 最佳时间：随含脂肪的餐食

**2. 亚精胺 (Spermidine)**
- 机制：诱导自噬，改善线粒体功能
- 益处：激活细胞自噬，动物研究延长寿命+25%
- 食物来源：小麦胚芽、大豆、发酵奶酪、蘑菇、纳豆
- 建议剂量：5-10mg/天
- 最佳时间：早晨或随早餐

**3. 白藜芦醇 (Resveratrol)**
- 机制：激活SIRT1（长寿基因），模拟热量限制
- 益处：激活去乙酰化酶，改善胰岛素敏感性
- 食物来源：红酒、红葡萄、蓝莓、黑巧克力
- 建议剂量：150-300mg/天或1杯红酒
- 最佳时间：晚上随晚餐

**4. Omega-3 (EPA/DHA)**
- 机制：抗炎、稳定细胞膜、神经保护
- 益处：降低炎症标志物（CRP ↓20-30%），改善大脑健康
- 食物来源：深海鱼（三文鱼、鲭鱼）、鱼油、藻油
- 建议剂量：1-2g EPA+DHA/天
- 最佳时间：随餐

**5. EGCG (Green Tea)**
- 机制：抗氧化，激活AMPK，抑制mTOR
- 益处：强效抗氧化，增强脂肪氧化，提升代谢率
- 食物来源：绿茶、抹茶
- 建议剂量：300-400mg EGCG（3-4杯绿茶）
- 最佳时间：早晨和下午（避免晚上）
