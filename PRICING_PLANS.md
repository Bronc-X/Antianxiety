# 定价方案详细说明

## 🎯 设计原则
- 参考 X 会员的分层逻辑：Free 负责体验、Pro 负责主要付费、先行版作为限量终身权益。
- 让权益逐层递进：从“只记录”到“AI 冷峻建议”，再到“深度生理信号 + 高速支持”。
- 所有价格与功能点均与最新版 README 对齐。

---

## 📋 三档方案

### Free版（免费入场）
- **价格：** ¥0
- **定位：** 体验冷峻方法论的最小版本，帮助用户建立记录习惯。
- **权益：**
  - 每日状态速记（睡眠 / 压力 / 情绪 3 项）
  - 7 日历史回顾
  - Web 端访问
  - 公开社区内容浏览
- **限制：** 无 AI 对话、无贝叶斯曲线、无个性化信息、无导出与提醒。

### 先行版（一次性 · 永久使用）
- **价格：** ¥99（限时）
- **定位：** 早期支持者专属，享有所有高级功能并永久解锁。
- **权益：**
  - Pro 全部权益
  - 深度生理信号分析（皮质醇 / 睡眠节律 / 代谢曲线）
  - 个性化信息推送（RAG：相关性 > 4.5/5）
  - AI 记忆系统 + 高速通道（对话永不重置）
  - 智能提醒（预测行为 + 提前推送）
  - 无限数据导出，Beta 功能优先体验
  - 专属 Onboarding & 私域社群

### Pro版（月度订阅）
- **价格：** ¥15 / 月
- **定位：** 主力订阅，满足大多数需要 AI 陪伴和贝叶斯分析的用户。
- **权益：**
  - Free 权益全部开放
  - AI 冷峻对话 + 贝叶斯信念循环完整图（信念 / 信心 / 身体机能）
  - 智能提醒（最小阻力习惯）
  - 个性化信息推送（每日精选）
  - 跨设备实时同步
  - 数据导出（每月 3 次）
  - 优先客服支持

## 💡 设计解读
1. **Free 极简**：只保留最基础的记录能力，确保体验入口但不稀释付费价值。
2. **Pro 主力**：参考 X Premium，提供“日常可感”的 AI 能力与图表输出。
3. **先行版**：参考 X Premium+，在性能、深度和服务上全面升级，并通过“一次性付费”制造稀缺感。

---

## 🔄 升级路径
1. Free → Pro：随时订阅/取消，7 天内可退款。
2. Free → 先行版：一次性购买，永久解锁全部高级功能。
3. Pro → 先行版：限时补差价升级。
